Welcome to the GitHub repo for Introductory R for Criminologists

This is the GitHub page where I store learning resources for R. The intended audience are criminology (or other social science) students.

Lab sessions for learning R data analysis programming

  • Week 1: Introduction to R
  • Week 2: Graphics with ggplot2
  • Week 3 Foundations for inference (confidence intervals)
  • Week 4: Hypothesis tests (comparing means)
  • Week 5: Relationships between two factors
  • Week 6: Linear regression
  • Week 7: Logistic regression
  • Week 8: Regression in the real world

Helpful video lectures and free online materials that can assist in following the previous lab sessions
